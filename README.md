## Features

### ⚡ Instant Gameplay
Deposit once, flip as many times as you want. Each flip settles immediately — your balance updates in real time. No payment modal between bets.

### 🔒 Provably Fair
Every coin flip uses **SHA-256 hash commitment**. Before you flip, the server publishes a hash. After the flip, the seed is revealed. You can verify any result independently — the server cannot change the outcome after your choice.

**How verification works:**
- Server generates a random `seed` and publishes `SHA256(seed + betId)` as the committed hash
- You pick a side and flip
- After the flip, the `seed` is revealed
- First character of hash < `8` = Heads, ≥ `8` = Tails
- Click **Verify** on any flip to check
